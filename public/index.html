Arquitetura NTC 4.0 — 13 Modulos

ModuloPesoDescricao
DNA25%Identidade molecular — fabricante + codigo + familia
TF15%Triangulacao Fiscal — NCM + EAN + CEST

CO10%Codigos Origem — CST + CEST + origem ICMS
AV10%Aplicacao Veicular — nasce do MOTOR, nao do veiculo
MC5%Metrologia Certificada — peso + dimensoes
EC5%Embalagem Comercial — qtd + tipo + material
BTA5%Base Tecnica Aplicada — motor + cilindrada + codigo
CC5%Codigos Cruzados — OEM/aftermarket/importadores separados
LG5%Logistica — prazo + frete + disponibilidade
IV5%Indexacao Visual — fotos reais confirmadas (min 4)
FI3%Financeiro — custo + venda + margem
FP2%Precificacao — estrategia de preco
